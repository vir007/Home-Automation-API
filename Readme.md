### Home Automation using Python/Django 

## Getting Started

Setup project environment with venv and pip.

```sh
$ python -m venv venv
$ venv/Scripts/activate
$ pip install -r requirements.txt
$ python manage.py migrate
$ python manage.py runserver
```
